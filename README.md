# Mira-NotePad
A notepad mod for Town of Us Mira (Among Us)

NotePad Mod v1.0.0

A simple notepad mod for Town of Us Mira. Opens a notepad window in the HUD where you can write notes during the game.

Features
- Toggle the notepad with a button in the HUD
- Multiline text with word wrap
- Click to place cursor
- Arrow keys, Home and End navigation
- Hold backspace/delete for fast deletion
- Clear button to erase all text
- Text is cleared on game start and lobby return
- Configurable text color and window skin via BepInEx config



Requirements
- Town of Us Mira
- BepInEx
